简单易用的HTTPServer服务器

2014-10-14

文章目录

  • 使用
  • 指定端口
  • 参考链接

使用

python -m SimpleHTTPServer

上面的命令就是在当前目录中建立HTTPServer服务器 如果当前目录下有一个叫 index.html 的文件名的文件，那么这个文件就会成为一个默认页，如果没有这个文件，那么，目录列表就会显示出来。 浏览器打开0.0.0.0:8000 或 localhost:8000 或 本机ip地址:8000即可使用。

指定端口

在8001端口建立HTTPServer

python -m SimpleHTTPServer 8001

如你所看到的就是如此的简单，但是这个功能在日常开发中很是常用。 ​
